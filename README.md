# 🖼️ IMAGE TO 3D GENERATOR

> **AI-Powered 3D Model Generation • Automated Meshing • High-Quality Textures**

---

## 🇺🇸 English

### 🌟 Overview
**Image to 3D Generator** is an automated tool that transforms 2D images into high-quality 3D models using AI. It integrates with **Meshy API** and **Instant Meshes** to provide a complete workflow from image input to textured 3D output.

### 🚀 Features
- 🤖 **AI-Powered Generation**: Convert any 2D image into a 3D model automatically.
- 🎨 **High-Quality Textures**: Generates detailed textures for realistic 3D models.
- 📐 **Automated Meshing**: Integrated with **Instant Meshes** for optimized quad-based topology.
- 📧 **Auto-Account Creation**: Built-in system for temporary email and account management for API access.
- 📊 **Progress Tracking**: Real-time progress bar and logging for the generation process.
- 📂 **Organized Output**: Automatically saves models in `.obj` and `.glb` formats with organized folder structures.

### 🛠️ Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Pedrohs1771/Image-to-3D-Generator.git
   cd Image-to-3D-Generator
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Requirements:**
   - [Python 3.10+](https://www.python.org/)
   - [Instant Meshes](https://github.com/wjakob/instant-meshes) (Included in the project)

### 📖 How to Use
1. Place your input image in the `input/` folder.
2. Run the main script:
   ```bash
   python main.py
   ```
3. Follow the prompts to select your image and generation quality.
4. Find your results in the `output/` and `result/` folders.

## 🇧🇷 Português

### 🌟 Visão Geral
**Image to 3D Generator** é uma ferramenta automatizada que transforma imagens 2D em modelos 3D de alta qualidade usando IA. Ele se integra com a **API Meshy** e o **Instant Meshes** para fornecer um fluxo de trabalho completo, desde a entrada da imagem até a saída do modelo 3D texturizado.

### 🚀 Funcionalidades
- 🤖 **Geração por IA**: Converta qualquer imagem 2D em um modelo 3D automaticamente.
- 🎨 **Texturas de Alta Qualidade**: Gera texturas detalhadas para modelos 3D realistas.
- 📐 **Malha Automatizada**: Integrado com **Instant Meshes** para topologia otimizada baseada em quads.
- 📧 **Criação Automática de Contas**: Sistema integrado para gerenciamento de e-mails temporários e contas para acesso à API.
- 📊 **Acompanhamento de Progresso**: Barra de progresso em tempo real e logs detalhados do processo de geração.
- 📂 **Saída Organizada**: Salva automaticamente os modelos nos formatos `.obj` e `.glb` com estruturas de pastas organizadas.

### 🛠️ Instalação

1. **Clone o repositório:**
   ```bash

   cd Image-to-3D-Generator
   ```

2. **Instale as dependências:**

### 📖 Como Usar
